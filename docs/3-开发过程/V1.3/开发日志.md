### 2026-06-25 | 会话 #1：V1.3 启动 — 14a/14b 实现 + 文件夹初始化

**作者**：CodeBuddy

**操作摘要**：
1. 创建 `docs/3-开发过程/V1.3/` 目录及 `开发日志.md`、`问题记录.md`
2. 更新开发阶段文档（阶段 14 → 开发中）
3. 实现 **14a 隐私保护**：
   - `HomeUiState.kt`：新增 `isAmountHidden` 字段（默认 true）
   - `HomeScreen.kt`：BalanceCard 右上角新增眼睛图标（Visibility/VisibilityOff），切换显示/隐藏金额
   - 隐藏态显示 `****`（首次为 `¥****`/`-¥****`/`+¥****`，后续简化为纯 `****`）
4. 实现 **14b 概览卡片内容调整**：
   - `TransactionDao.kt`：新增 `getGlobalSummary()`（一次性查询全局总收支）
   - `TransactionRepository.kt`：新增接口方法 `getGlobalSummary()`
   - `TransactionRepositoryImpl.kt`：实现 `getGlobalSummary()`
   - `GetHomeDataUseCase.kt`：`HomeSnapshot` 新增 `globalExpense`/`globalIncome`，flow 块内查询
   - `HomeViewModel.kt`：映射新增字段到 `HomeUiState`
   - `HomeUiState.kt`：新增 `globalExpense`/`globalIncome` 字段
   - `HomeScreen.kt`：BalanceCard 参数从 `totalIncome/totalExpense` → `globalIncome/globalExpense`，标签"本月支出/本月收入" → "总支出/总收入"
5. 14c 批量修改分类待讨论后单独开发
    6. **2026-06-25 二次编辑**：简化隐私隐藏格式，`formatHiddenAmount()` 统一返回 `"****"`，不再保留货币符号和正负号；支出/收入行隐藏态也统一为 `"****"`

**设计决策**：
- `isAmountHidden` 为 UI-only 状态（`remember { mutableStateOf(true) }`），不持久化，每次冷启动重置
- 仅概览卡片金额隐藏，汇总行、预算进度条、趋势图等区域不受影响
- `getGlobalSummary()` 为 suspend 一次性查询，复用在 UseCase 的 flow 块内
