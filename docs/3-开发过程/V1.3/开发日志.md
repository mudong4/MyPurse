### 2026-07-01 | 会话 #3：快捷退款 + V1.4 远期构想确认

**作者**：CodeBuddy

**操作摘要**：
1. **快捷退款功能（V1.4 6.4，提前实现）**：
   - 扩展 `Route.AddTransaction`：新增 `defaultAmount`、`defaultNote` 参数（均设默认值，不影响现有调用）
   - 扩展 `AddTransactionViewModel.initialize()`：支持预填金额、备注
   - `AddTransactionScreen`：新增 `defaultAmount`/`defaultNote` 参数；编辑模式支出记录底部显示「退款」按钮
   - `MyPurseNavHost`：编辑页退款导航（pop 当前编辑页 → navigate 新记账页），预填金额、备注="退款：yyyy-MM-dd"
2. **V1.4 远期构想确认**：
   - 同意取消统计页二级展开，改为单分类详情页承载
   - 统计图表增强、收入趋势图方向确认（详细内容待定）

**设计决策**：
- 退款入口选择：初版用长按触发，经讨论改为编辑页底部按钮。理由：右滑=删除 + 点击=编辑已占据两个手势，长按会造成手势竞争；编辑页提供更明确的意图表达空间，用户确认记录细节后再退款也更安全
- 退款分类不做自动匹配（退款分类名称不固定），让用户进入记账页后自选
- 金额预填后用户可在记账页修改（支持部分退款场景）
