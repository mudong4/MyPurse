## 阶段 5：预算设置页 (F)

**状态**：✅ 完成（2026-06-16）

### 功能清单

| 功能 | 说明 |
|---|---|
| 金额输入 | 数字键盘，¥ 前缀，占位符 0.00 |
| 快捷设置 | 3 个预设金额按钮（1000/5000/10000），点击快速填入，每个按钮旁编辑图标可自定义 |
| 保存按钮 | 保存预算，成功后弹出确认弹窗。金额为 0 时按钮置灰 |
| 不设预算 | 已设预算时显示红色按钮，点击清除预算值 |

### 设计要点

- 预算采用极简单一值存储（budget 表 id=1 唯一记录）
- 修改即覆盖，无历史追踪
- 首页预算进度条：<80% 蓝色，80-100% 橙色，≥100% 红色
- 无预算时首页显示引导文字"设置月度预算 >"
