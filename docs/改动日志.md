## 2026-06-16：时间导航移到底部 + 移除顶部导航栏

### 改动背景
用户反馈：
1. 顶部 TimeNavigator 不需要，时间导航（左右箭头+时间文本）应放在底部分类旁边
2. 底部独立的"时间选择器"项也不需要，点击时间文本弹出日历即可

### 改动内容

#### 1. 底部工具栏重构
新布局：`粒度 ▼ | < 时间文本 > | 分类级别 ▼`
- 时间导航（左/右箭头 + 时间文本）整合到底部，居中显示
- 时间文本点击弹出日历
- 移除底部之前独立的"时间选择器"（带下拉箭头的那个）

#### 2. 移除顶部 TimeNavigator
- 删除 `TimeNavigator` 组件及其 `HorizontalDivider`
- 列表内容直接从 Scaffold innerPadding 开始渲染

### 影响范围
- `TransactionListScreen.kt`：移除 TimeNavigator，重写 BottomToolbar

## 2026-06-16：分类简化 + 时间选择器重构

### 改动背景

用户反馈：
1. 分类下拉菜单过于复杂（树形展开），应简化为两个选项："按一级分类"、"按二级分类"
2. 底部日历按钮无用，点击时间选择控件直接弹出日历即可
3. 日历用数字不要英文

### 改动内容

#### 1. 分类逻辑简化
- `GroupMode` 从 `TIME / CATEGORY` 改为 `TIME / CATEGORY_L1 / CATEGORY_L2`
- 分类下拉菜单从复杂树形 → 三个选项："时间"、"一级分类"、"二级分类"
- 移除 `CategoryInfo`、`selectedCategoryL1/L2`、`allCategories` 等树形分类数据
- ViewModel 不再依赖 `CategoryRepository`，简化依赖
- `loadCategoryView` 根据 `CATEGORY_L1`/`CATEGORY_L2` 直接按对应级别分组

#### 2. 底部工具栏重构
- 从左到右：`粒度 ▼` | `分类级别 ▼` | `时间选择器 ▼`
- 时间选择器点击直接弹日历（年视图弹年选择器，月视图弹月选择器，日/周弹日期选择器）
- 移除独立日历图标按钮

#### 3. 日历纯数字显示
- `YearMonthPickerDialog` 中月份用 `1月~12月` 纯数字

### 影响范围
- `TransactionListUiState.kt`：GroupMode 枚举变更，移除 CategoryInfo
- `TransactionListViewModel.kt`：移除 CategoryRepository 依赖，简化分类逻辑
- `TransactionListScreen.kt`：BottomToolbar 重写，日历弹窗统一

## 2026-06-16：流水列表底部工具栏 + 分类分组重构（已废弃，见上方新版本）

### 改动背景

用户反馈：
1. 年/周/月/日 Tab 和分类筛选放在页面顶部，不符合手机单手操作习惯
2. 分类功能应该不仅用于"过滤"，还应支持"按分类分组"，改变年/月/周视图的分组维度
3. 缺少改动日志

### 改动内容

#### 1. 新增 `docs/改动日志.md`（本文件）
- 记录每次迭代的改动内容

#### 2. UI 布局重构：顶部控件移到底部
- **粒度选择**：从四个 Tab 按钮改为左下角下拉菜单，显示"年/月/周/日"
- **分类选择**：从顶部筛选栏改为底部下拉菜单，支持两级分类树形选择
- **日历选择器**：移到右下角，图标按钮触发
- **时间导航栏**（左右箭头+时间标签）：保留在页面顶部，便于快速切换时间
- 新增 **分组模式** 指示器：显示当前分组方式（按时间 / 按分类）

#### 3. 分类功能重构：从"过滤"变为"分组维度切换"
- 新增 `groupMode` 字段：`TIME`（按时间分组，默认） / `CATEGORY`（按分类分组）
- 选择分类后：年/月/周/日视图的数据分组方式从时间维度切换为分类维度
  - 年视图：12个月 → 按一级分类分组
  - 月视图：N个周 → 按一级分类分组
  - 周视图：7天 → 按一级分类（或二级分类）分组
  - 日视图：平铺 → 按一级分类（或二级分类）分组
- 分类树形结构：先选一级分类，展开后可选二级分类
- "全部分类" = 回到按时间分组模式

#### 4. UiState 改动
- 新增 `groupMode: GroupMode`（TIME / CATEGORY）
- 新增 `selectedCategoryL1: String?`、`selectedCategoryL2: String?`
- 新增 `allCategories: List<CategoryInfo>`（树形结构，含一级和二级）
- `availableCategories` 保留但语义变化（仅用于下拉列表构建）
- 移除 `categoryFilter: String?`（功能合并到 groupMode + selectedCategory）

#### 5. ViewModel 改动
- 新增 `setGroupMode(mode)` / `setCategoryGroup(catL1, catL2?)`
- `loadCategories()` 改为加载完整分类树
- 年/月/周/日视图的加载方法新增按分类分组的分支
- `setCategoryFilter` 移除，合并到新的分类选择逻辑

#### 6. Screen 改动
- 新增 `BottomToolbar` 组件（粒度下拉 + 分类下拉 + 日历按钮）
- 移除 `GranularityTabs` 顶部 Tab 行
- 移除 `CategoryFilterBar` 顶部筛选行
- `CollapsibleGroupHeader` 根据 groupMode 显示不同标签格式

### 影响范围
- `TransactionListScreen.kt`：大幅重构
- `TransactionListViewModel.kt`：重构加载逻辑
- `TransactionListUiState.kt`：新增字段和类型
- `docs/改动日志.md`：新增
