## 阶段 4：统计页 (C)

**状态**：🔲 待开发

### 整体架构

三个子视图，左右滑动切换：
- **构成分析**：钱花在哪？— 横向条形图（主力）+ 饼图（辅助）
- **趋势分析**：支出随时间变化？— 柱状图 + 折线图叠加
- **对比分析**：和上月比怎样？— 双柱对比图

### 时间维度选择器（三视图共享）

| 粒度 | 默认值 | 切换方式 |
|---|---|---|
| 日 | 今天 | 左右箭头逐天，点击弹日历 |
| 周 | 本周 | 左右箭头逐周，点击弹周选择器 |
| 月 | 本月 | 左右箭头逐月，点击弹年月选择器 |
| 季 | 本季 | 左右箭头逐季，点击弹季选择器 |
| 年 | 今年 | 左右箭头逐年，点击弹年份选择器 |

### 构成分析

| 区域 | 描述 |
|---|---|
| 支出/收入 Tab | 顶部切换 |
| 横向条形图 | 按金额从高到低，显示金额+占比% |
| 二级分类展开 | 点击一级展开二级明细，再次点击收起 |
| 饼图切换 | 右上角按钮切换 |
| 跳转入口 | 底部"查看该分类全部流水 >" |

### 趋势分析

| 区域 | 描述 |
|---|---|
| 图表模式 | 纯柱状 / 纯折线 / 叠加（默认），三按钮切换 |
| X 轴数据点 | 日=30天、周=12周、月=12月、季=8季、年=全部 |
| 预算线 | 虚线阶梯标注 |
| 悬浮提示 | 点击数据点弹出摘要（总支出、TOP3分类） |
| 数据摘要 | 均值、最高月份、最低月份、环比变化 |

### 对比分析

| 区域 | 描述 |
|---|---|
| 对比对象 | 上月 / 去年同月 / 自定义，默认上月 |
| 总支出对比 | 两行数字，增长红/下降绿 |
| 双柱图 | 本月实色 vs 对比月半透明，按金额从高到低 |
| 增减明细 | 各分类增减金额和百分比 |
| 二级对比 | 点击展开两个时期的二级明细 |

### 技术选型

- 图表库：Vico（Compose 原生）或 MPAndroidChart
- 聚合计算：Room `@Query` SQL 层完成
- 用例封装：`GetStatisticsUseCase`

## 阶段 5：预算设置页 (F)

**状态**：🔲 待开发

### 功能清单

| 功能 | 说明 |
|---|---|
| 月份切换 | 左右箭头切换任意月份 |
| 金额输入 | 数字键盘，为空显示"未设置" |
| "设为每月默认预算" | 勾选后批量应用到后续月份 |
| 保存按钮 | 修改历史月份时弹出确认提示 |
| 历史列表 | 各月预算，标注"已执行/未执行/自动沿用" |
| 智能沿用 | 新月份自动沿用最近一次手动设定的值 |

### 核心规则

- 预算按月独立存储
- 用户手动修改 → `isAutoGenerated = false`
- 新月份自动沿用 → `isAutoGenerated = true`
- 修改当前月不影响历史月份
- "本月起长期生效"仅更新 `isAutoGenerated = true` 的月份
