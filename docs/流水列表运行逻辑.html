为什么时间模式不崩？

  • 时间模式每个 group 代表不同的时间段（如"1月"的 rangeStart=Jan1，"2月"的 rangeStart=Feb1），rangeStart 天然不同 → key 不冲突
  • 分类模式所有 group 共享同一个时间窗口 → rangeStart 完全相同 → LazyColumn 检测到重复 key → 抛出异常崩溃

修复方案

  • 分类模式改用 group.label（如"餐饮"、"交通·早餐"）作为 key
  • 时间模式保持用 group.rangeStart
  • 折叠状态 expandedGroups 中的 Set 存储的也是对应的 key，需保持一致
